What is Python?

Python is called high level language because it writes hardware independent code (those programing language writes hardware dependent code called low level programming languages). As it is general purpose language, because it can make any kind of general purpose software.
It was created by Guido van Rossum during 1985-1990. Like Perl, Python source code is also available under the GNU General Public License (GPL)

      Why Python?

      The python language is one of the most accessible programming languages available because it has simplified syntax and not complicated like other language. Due to its ease of learning and usage, python codes can be easily written and executed much faster than other programming languages.

      Features of Python

      A. Python is Object-Oriented:-

      Python supports Object Oriented Programming(OOP) concept that encapsulate code within objects. All concepts in OOPs like data hiding, operator overloading, inheritance etc. can be well written in Python.

      B. Simple and easy-to-learn:-

      Python is a simple language with few keywords, simple structure and its syntax is also clearly defined. This makes Python a beginner's language.

      C. Interpreted and Interactive:-

      Python is processed at run time by the interpreter. We need not compile the Program before executing it. The Python prompt interact with interpreter to interpret the programs that you have writen.

      D.Extendable:-

      You can alsoadd low-level modules to the Python interpreter. These modules enable Programmer to add to or customize their tools to be more efficient. It can be easily integrated with C,C++,Java,COM,ActiveX,and COBRA

      E. Dynamic:-

      Python provides very high-level dynamic data typed and supports dynamic type checking. It also supports automatic garbage collection.

      F. GUI Programming and Database:-

      Python supports GUI applications that can be created and ported to many libraries and windows systems, such as Microsoft foundation Classes(MFC), Macintosh, and the X Windows system of Unix. Python also provides interfaces to all major commercial database.

      G. Portable:-

      Python can run on a wide variety of hardware and software platforms and has the same interface on all platforms. All variants of Windows, Unix, Linix and Macintosh are to name a few.

      H. Scalable:-

      Python provides a better structure and support dfor large Programs that shell scripting. It can be used as a scripting language or can be compiled to bytecode(intermediate code that is platform independent) for building large applications.

      I. Broad Standard Library:-

      Python's library is portable and cross platform compatible on UNIX, Linix, Windows and Macintosh. This helps in the support and development of a wide range of applications from simple text processing to browsers and compiex games.

      Identifiers

      A Python identifiers is a name used to idetify a variables, functions, class, module or any other object. Python is case sensitive and hence uppercase and lowercase letters are considered distinct. The following are the rules for naming an identifier in Python.

      1. Identifiers can be a combination of letters in lowercase(a to z) or uppercase(A to Z) or digits(0 to 9) or an underscore(_). For example- Python and python is different.
      2. Reserved keywords cannot be used as an identifier.
      3. Identifiers cannot begin with a degit. For example 2lower, 3lower etc are invalid identifiers.
      4. Special symbols like- !,@,#,$,%,^ etc. cannot be used in an identifiers. For example #name, @ram etc. are invalid identifiers.
      5. Identifiers can be of any length.

      Variables

      Variables are reserved memory locations to store values. Based on the data type of the variable, the interperter allocates memory and decides what can be stored in the reserved memory. Therefor, by assigning different types to variables, you can store integers, decimals or characters in these variables.
      Python variables do not need explicit declaratoion to reserve memory space. The decleration happens automatically when you assign a value to a variable. The equal sign(=) is used to assign values to variables. The operand to the left of the = operator is the name of the variable and the operand to the right of the = operator is the value stored in the variable.

      Example Program

      a=100 # an integer assignment
      b=500.0 # an floating point
      name="Ram" # A string
      print a
      print b
      print name

      Here, 100, 500.0, "Ram" are the values aassigned to a,b and name variable, respectively. Hence the output of the code is:-
